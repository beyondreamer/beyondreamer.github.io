Java进阶学习笔记（二）final关键字

Posted on 2021-03-30 In Java进阶
Symbols count in article: 358 Reading time ≈ 1 mins.

final关键字

  1. final是Java语言中的一个关键字
  2. 表示最终的不可变的
  3. final修饰的变量
    • 只能赋值一次
  4. final修饰的方法
    • 无法被覆盖，被重写
  5. final修饰的类
    • 无法继承
  6. final控制不了能不能调用

final修饰的引用

  1. final修饰的引用只能指向一个对象，并且它只能永远指向该对象
  2. 并且该方法执行过程中，该引用指向对象后，该对象不会被垃圾回收器回收，知道当前方法结束，才会释放空间
  3. 虽然final的引用指向对象A后，不能再重新向B,但是对象A内部的数据可以被修改

final修饰的实例变量

  1. final修饰的实例变量，系统不管赋默认值，要求程序员必须手动赋值
  2. 赶在系统赋默认值之前即可
  3. 一般添加static修饰为静态常量（常量名建议大写单词之间_连接）
  4. 常量和静态变量，都是存储在方法区，并且都是在类加载的时候初始化
  5. 常量一般都是公开的
